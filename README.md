### Installation
1. Clone the repository:
```sh
git clone https://github.com/mertcan-tas/vocabulary-app-flutter.git
```

2. Install dependencies:
```sh
flutter pub get
```

3. Run build_runner for code generation:
```sh
flutter pub run build_runner build
```

3. Run the app:
```sh
flutter run
```
